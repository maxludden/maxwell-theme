# Maxwell Carbon Color Theme

Maxwell Carbon Color Theme is a dark, low-contrast theme for Visual Studio Code with chartreuse accents. It is designed to provide a comfortable coding environment while still delivering clear visual cues. Colors are chosen to be harmonious and subdued, making code easier to read for extended periods.

## Getting Started

1. Copy the theme files into your VS Code extensions folder (`~/.vscode/extensions/` on Linux/Mac). Alternatively, package the theme using `vsce package` and install the resulting `.vsix` file via `code --install-extension <path-to-file>`.
2. After installing, open the Command Palette in VS Code and choose **Preferences: Color Theme**.
3. Select **Maxwell Carbon** from the list.
